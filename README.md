# Análise e Modelagem Preditiva de Notas de Filmes (TMDB)

Projeto desenvolvido para a disciplina de Inteligência Artificial.

## Objetivo

Desenvolver um modelo de regressão capaz de estimar a nota média dos filmes (`vote_average`) utilizando características disponíveis no conjunto de dados.

## Dataset

- Fonte: [Kaggle](https://www.kaggle.com/)
- Dataset: [TMDB Top 10000 Movies Updated 2026](https://www.kaggle.com/datasets/siddharthbhakta/tmdb-top-10000-movies-updated-2026)

## Tecnologias

- Python
- Google Colab
- Pandas
